# CorvoDelivery

CorvoDelivery is a Python-based project built with Streamlit to provide a seamless delivery management service.

## Dependencies

- Python 3.x
- Streamlit

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/diinoob/CorvoDelivery.git
   ```

2. Navigate to the project directory:
   ```bash
   cd CorvoDelivery
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the app:
   ```bash
   streamlit run app.py
   ```
